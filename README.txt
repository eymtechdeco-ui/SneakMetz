
# SneakMetz – Site vitrine statique

Fichiers inclus :
- `index.html` : page principale (hero, filtres, grille produits, FAQ)
- `styles.css` : styles modernes (noir / or)
- `script.js` : chargement + filtres + rendu des cartes
- `products.json` : catalogue (à éditer)
- `images/` : placez vos visuels produit ici
- `favicon.ico` : ajoutez votre icône si besoin

## Modifier le catalogue
- Ouvrez `products.json` et ajoutez/éditez vos paires.
- Champs : `brand`, `model`, `sku`, `sizes` (liste), `retail_price`, `market_avg`, `price`, `condition` (mettre "Neuf"), `image` (optionnel).

## Déploiement
- **Gratuit & facile** : glissez-déposez ce dossier sur **Netlify** ou **Vercel**.
- Ou GitHub Pages : poussez le dossier et activez Pages.

## Personnalisation
- Ajoutez vos images (≈800×600) dans `images/` et mettez le nom de fichier dans `image`.
- Personnalisez email/WhatsApp dans `script.js` et la section "À propos".
- Pour le SEO : changez `<title>` et `meta description` dans `index.html`.
